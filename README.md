# NextGen Wallcovering

Website for NextGen Wallcovering.

## Stack

Vite 7.3, React 19, TypeScript 5.9, Tailwind CSS v4, Framer Motion

## Setup

```bash
npm install
npm run dev     # http://localhost:5173
npm run build
```
